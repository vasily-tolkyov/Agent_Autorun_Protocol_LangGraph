# Changelog

English | [简体中文](CHANGELOG.zh-CN.md)

## 2.0.0

- First published LangGraph release of the workflow bundle.
- Preserves the same staged planning, autorun, and generator/critic verification model as `Agent_Autorun_Protocol`.
- Replaces the ACL-X file runtime with a local LangGraph runtime, checkpoints, and thread state.
- Adds automatic installation and configuration for the LangGraph environment.
- Packages the workflow as a standalone GitHub-ready distribution.
